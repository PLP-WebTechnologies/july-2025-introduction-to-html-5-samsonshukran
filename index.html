Programming Languages

These are some of languages that I have interacted with;

  • HTML

    • HTML is used to create the structure of the web pages. It defines elements like headings, paragraphs, images, links and more. It acts likje a skeleton of website telling the browser what to display. It works with CSS and JavaScript to build full Website

  • CSS

    • CSS is used to style and design HTML web pages. It controls the layout, colors, fonts, spacing, and animations on a website. While HTML structures the content, CSS makes it visually appealing. Styles can be written directly in the HTML file or separately in an external CSS file. CSS allows developers to create responsive and attractive web designs that work on all devices. It uses selectors to target HTML elements and apply styles. CSS is essential for web development and works with HTML and JavaScript to build complete and modern websites.

  • Python

    • yhton is a powerful beginner friendly programming language known for its clear and simple syntax. It is used in web development, artificial inteligence and software development. It allws one to build websites, analyse data or create AI models

  • Java

    • Java is a powerful, object-oriented programming language used for building desktop applications, mobile apps (especially Android), and large-scale backend systems. Unlike JavaScript, Java does not run in the browser but runs on the Java Virtual Machine (JVM), making it platform-independent. Java is known for its stability, performance, and scalability. It is widely used in enterprise systems, banking applications, and Android development. Java uses a strong syntax and structure, requiring you to define variables, data types, and class structures. It's a good choice for serious software development and is taught in many computer science programs.

  • JavaScript

    • JavaScript is a programming language used to make web pages interactive and dynamic. It runs in the browser and responds to user actions like clicks, input, or scrolling. With JavaScript, you can create slideshows, form validations, animations, games, and more. It can also communicate with servers to load data without refreshing the page (using AJAX). JavaScript is essential for frontend development and can also be used on the server side with Node.js. It's supported by all modern browsers and is a key technology in web development alongside HTML and CSS.
